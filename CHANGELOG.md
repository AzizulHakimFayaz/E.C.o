# Changelog

All notable changes to E.C.o will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-01
### Added
- High-performance FastAPI REST API backend (`server.py`) serving both API routes and static interface assets.
- Premium, feature-rich glassmorphic web dashboard for chat, databases, and system configuration.
- Real-time, interactive force-directed neural graph visualization of local and Neo4j memories (`vis-network`).
- Client-side voice dictation (STT) and voice synthesis (TTS) web speech integrations.
- Vector database (RAG) entries explorer and semantic data visualizer.
- System Configurations control panel mapping live to `settings.json`.
- Structural Git and GitHub Workflow Handbook (`docs/git_workflow.md`).
